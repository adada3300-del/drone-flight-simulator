가이드 북 · 시뮬레이터 사용 설명서

이 사이트는 지형·기상·기체 상태가 자동 비행에 미치는 영향을 비교하기 위한 교육용 3D 시뮬레이터입니다. 실제 비행 허가나 실제 기체의 안전성을 판정하는 프로그램은 아닙니다.

1. 기본 사용 순서

  1. 우측 상단 메뉴를 열어 지형과 날씨를 선택합니다.
  2. 바람·고도·배터리·적재 무게·경로를 설정합니다.
  3. 자동 비행 시작을 눌러 이륙과 경로 비행을 관찰합니다.
  4. 하단 계기판, 상단 비행 시간, 비행 기록으로 결과를 확인합니다.

2. 화면과 시점

  • 화면 드래그: 시점 회전
  • 마우스 휠 또는 손가락 확대: 화면 확대·축소
  • 파란 관: 계획된 자동 비행 경로
  • 드론과 경로 사이 점선: 현재 경로 이탈 정도

3. 환경 설정

평지산지해양

날씨는 맑음·비·눈·우박·번개를 선택합니다. 풍향은 기준 방위와 −90°~+90° 각도를 조합하여 정합니다.

4. 기상 피격 시스템

  • 우박 또는 번개에서 피격 확률·강도·판정 시간을 설정합니다.
  • 피격된 부품과 손상도는 구조 설계도에 빨간색으로 남습니다.
  • 손상은 임무가 끝나도 유지되며 다음 비행 성능에 영향을 줍니다.

5. 빠른 조건 버튼

  • 안전 조건: 낮은 바람과 충분한 배터리
  • 배터리 부족 조건: 저전력 비상 착륙 관찰
  • 드론 뒤집힘 조건: 강풍·돌풍에 따른 불안정과 전복 관찰

6. 구조 설계도

  • 왼쪽 화살표로 설계도를 열고 부품을 직접 선택합니다.
  • 투명도 게이지로 내부 구조 표시 정도를 조절합니다.
  • 드론 상태 초기화는 손상만 복구하며 드론 위치나 비행 단계는 바꾸지 않습니다.

7. 계기판과 버튼

  • 하단: 단계·고도·속도·배터리·안전 지수
  • 상단 중앙: 누적 비행 시간
  • 1×·2×·3×: 시뮬레이션 진행 속도
  • 비행 기록 확인: 과거 임무 결과 열기
  • 작은 i 버튼: 각 항목의 상세 설명

8. 손상 누적의 영향

모터·프로펠러 손상은 제어력과 속도를 낮추고, 배터리 손상은 전력 소모를 높입니다. 기체 손상은 안전 지수를 낮추며 심각한 손상은 다음 비행에서 비상 착륙을 유발할 수 있습니다.
