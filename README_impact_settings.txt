드론 환경 비행 시뮬레이터 - 기상 피격 설정 확장판

1. GitHub 저장소 최상위의 기존 index.html을 새 index.html로 교체합니다.
2. 우측 상단 메뉴에서 날씨를 '우박' 또는 '번개'로 선택합니다.
3. 피격 확률(%), 피격 강도(1~100), 피격 판정 시간, 시간 단위(초/분)를 설정합니다.
4. 자동 비행 시작을 누릅니다.
5. 지정 시간이 되면 설정한 확률로 한 번 피격 여부를 판정합니다.

강도 기준
- 번개: 강도 65 이상이면 기체 일부가 손상·분리되며 추락합니다.
- 우박: 강도 70 이상이면 반복 충격을 받으며 추락합니다.
- 기준 미만에서는 손상 후 자동 자세 보정을 시도하고 비행을 계속합니다.

설계도
- 피격 시 구조 설계도가 자동으로 열립니다.
- 피격 위치는 주황색 또는 빨간색으로 표시됩니다.
- 부품을 누르면 누적 손상도와 교육용 고장 확률을 확인할 수 있습니다.
- 투명도 조절 아래 '드론 상태 초기화' 버튼으로 손상 상태와 외형을 복원할 수 있습니다.

주의
- 모든 수치와 연출은 학교 탐구를 위한 교육용 단순화 모델입니다.
- 실제 드론의 안전성·고장률·비행 허가를 판정하지 않습니다.
- Three.js를 CDN에서 불러오므로 실행 시 인터넷 연결이 필요합니다.
