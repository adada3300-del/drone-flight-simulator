3D 드론 환경 비행 시뮬레이터 수정본

적용된 주요 기능
1. 드론 상태 초기화 버튼은 손상만 복구하며 현재 기체 위치와 비행 단계는 바꾸지 않음
2. 화면 상단 중앙에 누적 비행 시간 표시
3. 손상된 설계도 부품은 손상 정도와 관계없이 빨간색으로 표시
4. 손상 상태는 임무 종료 및 다음 시뮬레이션에서도 유지
5. 손상 상태를 브라우저 localStorage에 저장하여 페이지를 다시 열어도 유지
6. 누적 손상이 다음 비행의 제어력, 속도, 배터리 소모, 안전 지수에 반영
7. 심각한 핵심 부품 손상은 다음 비행에서 비상 착륙을 유발할 수 있음
8. 왼쪽 상단에 가이드 북 버튼 및 전체 사용 설명서 추가
9. 비행 기록에 누적 손상 수치 추가

GitHub Pages 적용 방법
- 기존 저장소의 index.html을 이 파일로 덮어씁니다.
- Commit changes를 누른 뒤 Pages 주소를 새로고침합니다.
- 이전 화면이 보이면 브라우저 캐시를 삭제하거나 강력 새로고침합니다.

주의
- 손상률과 고장 효과는 학교 탐구용으로 단순화한 교육용 값입니다.
- Three.js CDN을 사용하므로 실행 시 인터넷 연결이 필요합니다.
