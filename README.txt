3D 드론 환경 비행 시뮬레이터

1. index.html 파일을 GitHub 저장소의 최상위 폴더에 업로드합니다.
2. GitHub 저장소 Settings > Pages에서 main 브랜치와 /root를 선택합니다.
3. 공개된 주소를 열고 우측 상단 메뉴 버튼을 누릅니다.
4. 평지·산지·해양 지형과 맑음·비·눈 날씨를 선택할 수 있습니다.
5. 안전 조건, 배터리 부족 조건, 드론 뒤집힘 조건을 적용한 뒤 자동 비행 시작을 누릅니다.

주의
- 사진을 실제 3D 스캔한 모델이 아니라, 사진에서 확인되는 붉은 X자 동체·4개 모터·프로펠러 보호 가드·착륙 다리·전면 돌출부를 Three.js 도형과 절차적 질감으로 재구성한 모델입니다.
- Three.js를 CDN에서 불러오므로 실행 시 인터넷 연결이 필요합니다.
- 학교 탐구용 단순화 모델이며 실제 드론의 비행 안전이나 허가를 판정하지 않습니다.
