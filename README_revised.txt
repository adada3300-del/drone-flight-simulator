3D 드론 환경 비행 시뮬레이터 수정본

1. GitHub 저장소의 기존 index.html을 이 ZIP 안의 index.html로 덮어씁니다.
2. Commit changes를 누릅니다.
3. GitHub Pages가 갱신된 뒤 새로고침합니다.
4. 이전 화면이 계속 나오면 브라우저 캐시를 삭제하거나 강력 새로고침합니다.

주요 수정 사항
- 삼각 비행 경로를 세로형 정삼각형으로 수정
- 산지의 봉우리, 능선, 바위, 침엽수 표현 강화
- 산지 지표 높이에 따라 경로와 목표 고도가 변하도록 유지
- 드론과 가장 가까운 경로 지점 사이에 점선 추가
- 점선 색상: 경로 오차가 작으면 초록, 보통이면 노랑, 크면 빨강
- 왼쪽 아래 시점 이동 조이스틱 추가
- 태풍 조건 제거
- 바람 선을 더 선명하게 만들고 드론 주변에서는 자동으로 피하도록 수정
- 풍향을 동서남북 기준 방향과 0~90도 각도로 분리
- 각도 게이지에 30도, 45도, 60도, 90도 표시
- 설정 항목과 계기판에 클릭 가능한 i 도움말 추가

주의
- Three.js를 CDN에서 불러오므로 실행할 때 인터넷 연결이 필요합니다.
- 이 프로그램은 학교 탐구용 단순화 모델이며 실제 비행 안전이나 허가를 판정하지 않습니다.
